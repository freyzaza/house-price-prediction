Cara Menjalankan Aplikasi Streamlit - Prediksi Harga Rumah Jabodetabek
=======================================================================

1. Pastikan semua file sudah tersedia 

Struktur folder yang disarankan:
├── daftar-harga-rumah-jabodetabek.csv   <- Dataset (pastikan sesuai)
├── model.pkl                            <- File model pipeline (sudah termasuk preprocessing)
├── app.py                               <- File utama aplikasi Streamlit

2. Pastikan semua library yang dibutuhkan sudah terinstall

3. Di terminal / command prompt / VS Code: "streamlit run app.py"
